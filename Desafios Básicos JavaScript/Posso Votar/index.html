Posso Votar

Escreva um programa para ler e imprimir no console o ano de nascimento de uma pessoa e escrever uma mensagem que diga se ela pode votar este ano ou não. (não é necessário considerar o mês em que ela nasceu).

01 - Se idade for maior que 18 e menor que 70: Voto Obrigatório
02 - Se idade estiver entre 16 e 17 ou for maior que 70: Voto Opcional
03 - Se idade for menor que 16: Não Pode Votar
